FastAPI Integrations

Using FastAPI, SQLAlchemy and PostgreSQL. Performing user Authentication and Authorization using JWT along with Google Oauth 2.0, integration with Zoom API for scheduling virtual meetings.

I developed this project inspired by the need to integrate FastAPI with other technologies like PostgreSQL and Google Oauth 2.0

While doing this project I had to learn a lot about authentication using JWT, cookies and sessions. Also, I learned about PostgreSQL and how to integrate it with ORM's like SQLAlchemy.

Naval Battle

Naval Battle game made using Java, Javafx, css applying OOP, EDP concepts and serializable files to save the progress of the game.

Naval Battle game is an interactive desktop application inspired by the classic battleship game. Designed with a modern twist, this JavaFX-based application provides players with an engaging experience, allowing them to develop a strategy, attack and conquer their opponent's fleet.
